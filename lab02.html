Тема, мета, посилання

Тема: Стильове оформлення елементів в HTML. Каскадні таблиці стилів. Селектори.

Мета: Опанувати на практиці основні способи підключення CSS до HTML-документу.

Посилання:

  • Репозиторій WEB-застосунку: Lord of Books (Букініст)
  • Жива сторінка WEB-застосунку: Lord of Books (Букініст)
  • Репозиторій звітів: Звіт лабораторних робіт
  • Жива сторінка звітів: Звіт лабораторних робіт

Варіанти підключення CSS до HTML-документу

  1. Варіант з підключенням зовнішнього файлу

                    
                  <link rel="stylesheet" href="style.css">

  2. Варіант використання inline стилів

                    
                  <div style="color: red;">Назва елементу</div>

  3. Внутрішні стилі html сторінки

                    
                  <style>
                    body {
                      background-color: #f0f0f0;
                    }
                  </style>

Селектори тегів

  1. Селектор тегу застосовує стилі до всіх елементів певного типу.
  2. Синтаксис: назва_тегу { властивість: значення; }

Приклад HTML-коду WEB-застосунку з використанням семантичних тегів:

                
                  <head>
                    <meta charset="utf-8" />
                    <meta name="viewport" content="width=device-width, initial-scale=1" />
                    <title>Мій перший веб-сайт</title>
                    <link rel="stylesheet" href="css/styles.css" />
                  </head>
                  
                  <body>
                    <header>
                      <h1>Ласкаво просимо на мій веб-сайт!</h1>
                    </header>
                    
                    <nav>
                      <ul>
                        <li><a href="#">Головна</a></li>
                        <li><a href="#">Про мене</a></li>
                        <li><a href="#">Контакти</a></li>
                      </ul>
                    </nav>
                    
                    <main>
                      <section>
                        <h2>Про мене</h2>
                        <p>Це мій перший веб-сайт, створений з використанням HTML та CSS.</p>
                      </section>
                    </main>
                    
                    <footer>
                      <p>Авторські права © 2025</p>
                    </footer>
                  </body>

Приклад з CSS

                
                  header {
                    background-color: #4CAF50;
                    color: white;
                    text-align: center;
                    padding: 10px 0;
                  }

Селектори класів

  1. Селектор класу застосовує стилі до всіх елементів з певним класом.
  2. Синтаксис: .назва_класу { властивість: значення; }

Приклад HTML-коду з використанням класів:

                
                  <div class="highlight">Цей текст буде виділений.</div>
                  <p class="highlight">Цей абзац також буде виділений.</p>

Приклад з CSS

                
                  .highlight {
                    background-color: yellow;
                    font-weight: bold;
                  }

Селектори ID

  1. Селектор ID застосовує стилі до елемента з певним ID.
  2. Синтаксис: #назва_id { властивість: значення; }

Приклад HTML-коду з використанням ID:

                
                  <div id="header">Це заголовок сторінки.</div>

Приклад з CSS

                
                  #header {
                    background-color: blue;
                    color: white;
                    padding: 15px;
                    text-align: center;
                  }

Інші селектори

  1. Селектор атрибуту: вибирає елементи з певним атрибутом. Синтаксис: [атрибут] { властивість: значення; }
  2. Комбіновані селектори: дозволяють поєднувати різні типи селекторів. Синтаксис: назва_тегу.клас { властивість: значення; }
  3. Псевдокласи: застосовують стилі до елементів у певному стані. Синтаксис: назва_тегу:hover { властивість: значення; }

Приклад з CSS

                
                  /* Селектор атрибуту */
                  a[target="_blank"] {
                    color: red;
                  }
                  
                  /* Комбінований селектор */
                  p.intro {
                    font-size: 18px;
                    font-weight: bold;
                  }
                  
                  /* Псевдоклас */
                  button:hover {
                    background-color: green;
                    color: white;
                  }

Advanced CSS

  1. Медіа-запити: дозволяють застосовувати різні стилі в залежності від розміру екрану. Синтаксис: @media (max-width: 600px) { назва_тегу { властивість: значення; } }
  2. Flexbox: забезпечує гнучке розташування елементів у контейнері. Синтаксис: .container { display: flex; justify-content: center; align-items: center; }
  3. CSS Grid: дозволяє створювати складні макети сторінок. Синтаксис: .grid-container { display: grid; grid-template-columns: repeat(3, 1fr); gap: 10px; }

Приклад з CSS

                
                  /* Медіа-запити */
                  @media (max-width: 600px) {
                    body {
                      background-color: lightgray;
                    }
                  }
                  
                  /* Flexbox */
                  .container {
                    display: flex;
                    justify-content: center;
                    align-items: center;
                  }
                  
                  /* CSS Grid */
                  .grid-container {
                    display: grid;
                    grid-template-columns: repeat(3, 1fr);
                    gap: 10px;
                  }
