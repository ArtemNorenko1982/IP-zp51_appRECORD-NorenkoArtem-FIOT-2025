Тема та мета лабораторної роботи №1

Тема: Структура HTML-документа та створення інформаційного WEB-застосунку: теги, атрибути, зображення, списки, форми, посилання та робота з Git/GitHub.

Мета: Навчитися створювати семантичну структуру HTML-сторінки, використовувати основні елементи розмітки та організовувати роботу з кодом у репозиторії GitHub.

У роботі розроблено структуру WEB-застосунку «Lord of Books (Букініст)», описано його предметне середовище та наведено приклади HTML-коду основних елементів сторінки.

Опис предметного середовища та бізнес-логіки

  • Тема WEB-застосунку

    Lord of Books (Букініст)

    В сучасному світі, де інформаційні технології стрімко розвиваються, обмін знаннями та ресурсами стає все більш важливим аспектом повсякденного життя. Однією з таких ресурсів є книги, які відіграють ключову роль у розвитку культури читання та освіти.

  • Мета створення

    Проєкт "Lord of Books" (Букініст) є веб-застосунком, який спрямований на створення онлайн-платформи для обміну книгами серед користувачів. Ідея полягає в тому, щоб надати людям можливість ділитися своїми улюбленими книжками, знаходити нові для читання та сприяти розвитку спільноти читачів.

  • Опис предметного середовища

    Платформа забезпечечить користувачам можливістю реєструватися, створювати профілі, додавати книги до своїх віртуальних бібліотек, обмінюватися книгами з іншими користувачами та переглядати доступні книжки. Користувачі зможуть залишати відгуки та оцінки для книг, що сприятиме формуванню спільноти читачів.

  • Функціональні вимоги

    • Авторизація
    • Каталог наявних книжок.
    • Пошук книжок.
    • Рейтинг книжок.
    • Статистика обміну книжками.
    • Можливість залишати відгуки та оцінки.

  • Нефункціональні вимоги

    • зручний інтерфейс
    • швидкодія
    • безпека даних користувачів
    • масштабованість

Структура html-документа (semantic design)

                
                  <header>
                    <div class="header__logo">...</div>
                      <nav>...</nav>
                    <div class="auth">...</div>
                  </header>
                  
                  <main>
                  
                    <section class="hero">...</section>
                  
                    <section class="page-wrapper">
                      <article>...</article>
                      <article>...</article>
                    </section>
                  
                  </main>
                  
                  <footer>
                    <section class="footer__col">...</section>
                  </footer>

Структура WEB-застосунку побудована з використанням семантичних HTML5-тегів <header>, <nav>, <main>, <section>, <article> та <footer>.

У звітному HTML-документі додатково наведено приклади використання таблиць, зображень, маркованих списків та елементів форм.

HTML-код зображень

                
                  <img src="src/pic1.jpg" alt="avatar" class="avatar" />

Пояснення атрибутів тега <img>:

  • src - шлях до зображення
  • alt - альтернативний текст для зображення
  • class - клас для застосування стилів CSS

HTML-код списків

                
                  <ul>
                    <li>Перший елемент списку</li>
                    <li>Другий елемент списку</li>
                    <li>Третій елемент списку</li>
                  </ul>

Пояснення тегів списку:

  • <ul> - маркований список
  • <li> - елемент списку

HTML-код форм

                
                  <form action="/submit" method="post">
                    <label for="name">Ім'я:</label>
                    <input type="text" id="name" name="name" />
                    <input type="submit" value="Відправити" />
                  </form>

Пояснення тегів форми:

  • <form> - контейнер для елементів форми
  • <label> - мітка для елемента форми
  • <input> - елемент введення даних
