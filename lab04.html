Тема та мета лабораторної роботи №4

Тема: Основи JavaScript: змінні, розгалуження, цикли, функції, масиви та валідація HTML-форми.

Мета: Навчитися створювати окремі JavaScript-програми, працювати з введенням користувача, виконувати обчислення, обробляти масиви й перевіряти дані форми.

Для кожного завдання створено окремий файл js/lab04_taskXX.js. Результати виводяться через alert(), у Console браузера або безпосередньо на сторінці.

Завдання 1. Перевірка введеного числа

Зберегти введене через prompt() значення у змінну value, вивести його шаблонним рядком у Console та визначити, чи число додатне, від’ємне або дорівнює нулю.

Файл програми: js/lab04_task01.js

Завдання 2. Визначення пори року

Запитати число від 1 до 4 та за допомогою switch-case записати у result відповідну пору року. Передбачити гілку default.

Файл програми: js/lab04_task02.js

Завдання 3. Авторизація Admin та User

Запитати логін і пароль. Для користувачів Admin та User перевірити відповідні паролі, обробити скасування введення та невідомий логін.

Тестові дані: Admin / admin123, User / user123

Файл програми: js/lab04_task03.js

Завдання 4. Станція продажу ремонтних дроїдів

Функція makeTransaction(quantity, pricePerDroid) обчислює загальну вартість замовлення та повертає повідомлення про покупку.

Файл програми: js/lab04_task04.js

Завдання 5. Перевірка повідомлення на спам

Функція checkForSpam(message) без урахування регістру перевіряє наявність слів spam або sale та повертає булеве значення.

Файл програми: js/lab04_task05.js

Завдання 6. Фільтрація масиву

Функція filterArray(numbers, value) створює та повертає новий масив із числами, більшими за задане значення.

Файл програми: js/lab04_task06.js

Завдання 7. Обробка та сортування масиву

Створити масив заданого користувачем розміру, знайти максимальний парний елемент і мінімальний елемент із парним індексом, поміняти їх місцями та відсортувати результат методом вставки.

Файл програми: js/lab04_task07.js

Завдання 8. Двовимірний масив

Створити двовимірний масив випадкових чисел, сформувати окремі масиви додатних і від’ємних чисел та замінити третій додатний елемент введеним від’ємним значенням.

Файл програми: js/lab04_task08.js

Завдання 9. Валідація HTML-форми

Форма перевіряє обов’язкові поля, ціле та дійсне числа, дату у форматі DD.MM.YYYY і однаковість паролів.

Файл програми: js/lab04_task09.js
