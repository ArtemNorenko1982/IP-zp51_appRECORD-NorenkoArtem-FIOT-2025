Лабораторна робота №7

Тема: МОДУЛЬНОСТЬ КОДУ. NODE.JS. JSON. WEB STORAGE API. АСИНХРОННІСТЬ. ПРОМІСИ. HTTP-ЗАПИТИ. REST API. AJAX. КРОСДОМЕННІ ЗАПИТИ. ПАГІНАЦІЯ. CRUD.

Мета: опанувати збереження стану у Web Storage, асинхронні операції та Promise, створення галереї, модального перегляду, пошуку даних і пагінації.

Структура рішення

Робота складається з п’яти окремих інтерактивних модулів. Форма використовує localStorage; таймер і генератор повідомлень демонструють асинхронність; галерея створюється з JSON-подібного масиву; пошук імітує REST-запит через Promise, loader та пагінацію.

Завдання 1. Таймер зворотного відліку

Пояснення

Користувач обирає майбутню дату. Після запуску setInterval() щосекунди обчислює різницю між кінцевою і поточною датою. Значення форматуються через padStart(). Під час відліку елементи керування заблоковані.

00Days
00Hours
00Minutes
00Seconds

Оберіть дату в майбутньому.

Файл: js/lab07_timer.js

Завдання 2. Форма зворотного зв’язку та Web Storage API

Пояснення

Подія input делегована формі. Актуальні значення зберігаються в localStorage під ключем feedback-form-state і відновлюються після перезавантаження. Успішний submit очищає форму та сховище.

Дані ще не відправлено.

Файл: js/lab07_form.js

Завдання 3. Генератор промісів

Пояснення

Після submit створюється Promise. Через задану затримку він переходить у стан fulfilled або rejected. Методи then(), catch() та finally() обробляють результат і показують повідомлення.

State

Очікується створення промісу.

Файл: js/lab07_promises.js

Завдання 4. Галерея активного відпочинку на роликах

Пояснення

Розмітка карток генерується в JavaScript із масиву об’єктів. Єдиний обробник кліку встановлено на контейнері галереї, тобто використовується делегування. Клік по зображенню відкриває велику версію у власному модальному вікні.

    Файл: js/lab07_gallery.js

    Завдання 5. Асинхронний пошук зображень і пагінація

    Пояснення

    Форма надсилає пошуковий запит до локального каталогу через асинхронну функцію, яка повертає Promise аналогічно REST API. Під час очікування показується loader. Результати розбиваються на сторінки, а кнопки пагінації перемикають поточну сторінку.

    Введіть пошуковий запит. Наприклад: ролики, парк, друзі або скейтпарк.

      Файл: js/lab07_search.js

      У лабораторній роботі реалізовано збереження стану у localStorage, асинхронні таймери, Promise, динамічну галерею, модальне вікно, пошук, loader і пагінацію. Кожний функціональний блок винесено в окремий JavaScript-модуль.
